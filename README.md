# laserFPS

laserFPS is a browser-based first-person laser shooter built with Next.js, React Three Fiber, Three.js, and Zustand.

Aim with the mouse, swap between beam and pulse weapons, dash through the arena, and survive as enemy waves escalate.

## Play locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Build

```bash
pnpm build
```

## Controls

- Click the game to lock the pointer and start aiming.
- Use `WASD` to move around the arena.
- Hold `Shift` to sprint.
- Press `Space` to dash in your movement direction.
- Move the mouse to look around.
- Press `1` for the beam weapon.
- Press `2` for the pulse weapon.
- Hold the left mouse button to fire the beam, or click to fire the pulse.
- Survive as long as you can.
